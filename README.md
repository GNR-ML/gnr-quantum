\# GNR Quantum Prototype



This project is an early open-source prototype for exploring small Hückel Hamiltonians with adjustable alpha and beta parameters, inspired by graphene nanoribbon systems.



\## What it does

\- Builds toy Hückel Hamiltonians

\- Solves them classically

\- Estimates HOMO, LUMO, and bandgap

\- Demonstrates simple structure-aware parameterization

\- Compares Jordan–Wigner qubit counts with compact encoding



\## Why it matters

This prototype is a small, reproducible benchmark workflow for students and researchers interested in the intersection of computational chemistry and near-term quantum methods.



\## Current features

\- Python source modules

\- Test script

\- Lightweight Streamlit demo



\## Future directions

\- Add graph-based descriptors

\- Add learned alpha and beta prediction

\- Add VQE-based experiments

\- Expand toward selected graphene nanoribbon model systems
